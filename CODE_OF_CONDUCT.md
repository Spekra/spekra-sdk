# Code of Conduct

We're all here to build great software. To make that happen, we expect everyone participating in the Spekra community to:

1. **Be respectful** - Disagree with ideas, not people
2. **Be constructive** - Criticism is fine, but offer solutions
3. **Be inclusive** - Welcome newcomers and help them learn
4. **Be professional** - Keep discussions focused and productive

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling or inflammatory comments
- Spam or off-topic content
- Publishing others' private information

## Enforcement

Violations may result in:
1. Warning
2. Temporary ban
3. Permanent ban

## Reporting

If you experience or witness unacceptable behavior, [open a GitHub issue](https://github.com/spekra/spekra-sdk/issues/new) and tag @spekra.

All reports will be reviewed and investigated promptly and fairly.

## Attribution

This Code of Conduct is adapted from common open-source practices.
